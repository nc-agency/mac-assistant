# Mac Assistant Ressourcen

Dieser Ordner enthält alle benötigten Ressourcen für die Erstellung des MacAssistant DMG-Installationspakets.

## Inhalt

- `icons/` - Symbole für die Anwendung und das DMG
  - `AppIcon.icns` - Hauptsymbol der Anwendung
  - `*.png` - Icon-Dateien in verschiedenen Größen
- `dmg_background.png` - Hintergrundbild für das DMG-Installationsfenster
- `templates/` - Vorlagen für die DMG-Erstellung

## Verwendung

Diese Dateien werden vom `create_dmg.sh`-Skript im Verzeichnis `/scripts` verwendet, um die DMG-Installationsdatei zu erstellen.
